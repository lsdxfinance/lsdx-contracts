# LSDx Contracts

## Compile

Cloning repository and install dependencies:

```sh
$ git clone <repository-url>

$ cd lsdx-contracts

$ yarn
```
Compile contracts:

```sh
# Use `hardhat-shorthand`:
$ hh compile

# Or, use `yarn`:
$ yarn run hardhat compile

# Or, use `npx`:
$ npx hardhat compile
```

## Run Test Cases

```sh
$ hh test
```
