
## Deploy

### Prepare `.env` 

With same keys to `.env-example`

### Deploy LsdCoin

```sh
$ hh run scripts/v2/deployLsdxV2Contracts.ts --network <mainnet/goerli>

# Etherscan verify
$ hh verify --network <mainnet/goerli> <address>
```
